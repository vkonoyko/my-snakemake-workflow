# Использование LLM при выполнении учебных заданий

## Обзор
В ходе выполнения заданий использовал DeepSeek как помощника.

## Сценарии использования

### 1. Пошаговое планирование
LLM помогал разбивать сложные задания на шаги.

### 2. Отладка ошибок
При ошибках LLM давал конкретные решения.

## Примеры из работы над проектом

### Пример: Исправление веток Git
**Проблема:** Ветка main была новее feature.

**Мой запрос:**
```
Как исправить чтобы main был старше feature?
```

**Ответ LLM:**
```
git checkout main
git reset --hard 5c9051b
git push --force origin main
```

## Выводы

1. LLM полезен для пошагового планирования
2. Нужно проверять сгенерированный код
3. Лучше работать через диалог с уточнениями

## Использованная модель
- DeepSeek (через веб-интерфейс)
